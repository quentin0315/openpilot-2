돌비스성실멤버 1:1 채팅

부팅화면은 3가지가 있습니다. 
-----------------------------------------------------------------------------------------------
1. 러프로 부팅화면(12초 정도 진행되며 여러번 글이 올라왔습니다)
76번글, 223번글, 324번글 입니다. 

-----------------------------------------------------------------------------------------------

2. /system/media/ 에 있는 bootanimation.zip의 그림파일 입니다.
(30~33초 정도진행됩니다. 기존의 콤마그림에 큰원이 돌아가는 화면입니다)

1) bootanimation 파일을 받아서 압축을 풀면 위의 폴더와 텍스트 파일이 하나 나옵니다. 
텍스트 파일을 여시면 
1920 1080 30                      영상의 크기, 재생프레임 수
c 1 0 part0                        첫번째 숫자 1은 이 이미지들을 한번재생한다는 의미
c 0 0 part1                        첫번째 숫자 0은 무한재생한다는 의미
c 1 30 part2
두번째 숫자는 끝마치고 딜레이를 준다는데 뭔말인지 모르겠습니다. 
저는
1920 1080 30                    
c 1 0 part0                        
c 1 0 part1                        
c 1 0 part2
으로 수정했습니다. 

2) 넣고 싶은 동영상을 정합니다. 30초에서 35초가 정도가 적당합니다.
곰플레이어와 포토스케이프를 사용해서 작업했습니다. 

우선 영상을 곰플레이어로 열고 재생화면에서 마우스 우클릭, 영상, 영상회전 180도 를 선택합니다.
(재생되는 동영상, 그림이 거꾸로 되야 이온에서 똑바로 나옵니다)

F7 을 누르면 주황색 테두리를 만들어놓은 제어창이 뜨고 
고급화면 캡쳐(빨간네모)를 누르면 바로 위에 창이 나옵니다. 
연속저장간격을 수정하시고
연속저장프레임을 수정합니다. 
저는 0.03 999로 했습니다. 

동영상 재생(녹색테두리)하시고 연속화면저장(녹색테두리) 누르시면 영상이 사진으로 저장됩니다. 

저는 550장 정도 그림 파일을 사용했습니다. 

적당히 part0, part1, part2에 분배하시면 됩니다.
(파일이름을 깔끔하게 수정하고 싶으시면 포토스케이프 사용하시면 됩니다. 

일괄편집, 좌측 폴더에서 보이는 파일 전부 선택해서 가운데로 드래그, 우측 상단의 모든 사진 저장(변환) 누르시고
  

다른 이름으로 바꿔서 저장합니다. 
파일명은 번호 형식 4자리 

앞쪽에 0 입력하고 나머지에 지정하면 원본과 비슷하게 다섯자리 숫자가 차례대로 이름이 됩니다. 


다 넣으셨으면 압축을 해야 하는데 반디집으로 할 때 상세설정보기 누르시고 압축방법에 압축하지 않음을 선택하고 압축을 해주셔야 합니다. 


3) 파일이 준비되었으면 /system/media 폴더에 파일을 교체해주시면 되는데 
그 전에 워크벤치 가셔서 
mount -o rw,remount /system     엔터 

해주셔야 저기 폴더에 복사해서 넣으실 수 있습니다. 

글이 길어서 헷갈리시면 답글 주시면 설명을 또 드리겠습니다. 
* 주의하실점은 part0 등등 폴더에 그림파일 말고 다른 폴더나 파일이 있으시면 제대로 진행되지 않습니다.
* 복사하 때 권한 문제가 발생할수 있으니 마우스 우클릭으로 복사하기해서 권한설정을 하시는게 좋습니다
아래 댓글에 내용이 나와있습니다

----------------------------------------------------------------------------------------------------

3. 흰색 아래 진행바가 뜨는 콤마 스피너 부분
(12~15초 진행됩니다)
1) openpilot/selfdrive/assets 폴더에서 아래파일을 원하는 파일로 대체한다.
   img_spinner_comma.png
   img_spinner_track.png
   png파일의 크기는 848*848입니다. 가운데 들어가는 거라 마크 같은 거면 배경 거의 안보이게 꽉차게 만드셔야 합니다. 
   저는 포토샵이 없어서 투명배경 귀찮아서 ~~comma.png 파일을 검은배경의 사진을 넣었더니 track 돌아가는것이 
   그냥 없어졌네요(원래 아래 진행 흰줄이 있어서 없애고 싶었는데 잘됐다 생각합니다)



2) 다 넣으셨으면 워크벤치 가셔서 cd /data/openpilot/selfdrive/ui/spinner/    엔터
   make  엔터

이러면 스피너 부분이 바뀌게 됩니다.
